     주식 데이터를 리스트로 만드는 파이썬

   실시간으로 주식현재가 파악하기

  현재 진행중인 파이썬을 이용한 '주식시장 확인' 프로젝트입니다.(미완성)

  (2022.01.15~ 2022.06월 현재까지 진행중)


  주식시장을 실시간으로 가격하고 매물을 할수있게 체크화는 프로그램이며 일정가격이하로 떨어지거나 예상가격이상으로 올라가게 될 경우에 알림과 함께 스크린샷으로 확인이 가능하도록 설계 하는 것을 목표로 제작중임.


  먼저 데이터를 받아올 서버 데이터를 선택해야하는데 초기에는 여러 사이트들에서 데이터를 가져오는것을 시도하였음.


  특정 주식에 대한 일일 최고가와 최저가를 텍스트 형식으로 받아오는데 성공하였고, 현재 받아온 데이터들을 표현하는 방식에 대해서 고민을 하고있는중입니다.


  but1

  특정 서버에서 데이터를 가져오는데 오류가 나거나 값이 제대로 입력이 되지않은 문제점들이 나와서 그중에서 네이버 주식시장을 이용하기로 결정을 하였음.


  but2

  네이버 주식시장에서 데이터를 가져오면서 특정 시간대에서 데이터를 가져오는것이 늦춰지거나 갱신이 조금 느리게 되는 경우가 발생하였음.


  Problem1

  현재 실시간으로 서버를 통해서 해당되는 주식의 현재가를 받아서 텍스트로 표현하는데 까지는 성공 하였으나 무엇보다도 현재가가 실질적으로 반영되는데 약간의 시간차이가 바로바로 반영되는 경우도 되지만 길면 10초정도 느리게 값을 받아오는 문제점이 있음.


  Problem2

  특정 번호에 대한값이 잘못 데이터를 전달받아서 엉뚱한 주식코드의 데이터를 받는 문제가 발생하엿음.


  Problem3

  데이터 코드값들을 잘못받아서 너무나도 많은 데이터값들을 한꺼번에 저장하다보니 렉이 발생하면서 프로그램이 종료되는 문제가 발생하엿음.


  Future

  해결해야 할 과제들및 진행방향


  방향

  1.데이터값들을 효율적으로 나열하기 위한 최적화된 리스트를 작성


  2.데이터값들간의 충돌이 일어나지않게 제대로된 코드를 작성


  3.데이터값들의 효율적으로 나타내기 위한 그래프및 표 제안하기


  (목표 및 잡담)


  일단은 최적화된 프로그램을 만들기 위한 노력을 기울여야 하지만 무엇보다도 중요한건 디자인과 간편성을 기반으로 제작을하는 것을 목적으로 해야할것같다.

  기술적으로나 디자인적으로나 아직부족한 부분들은 많고 메꿔나가야할 곳도 많기에 노력을 충분히 기울여야할것이다.
